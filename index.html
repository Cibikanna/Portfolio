Minesweeper


This program implements a simple Minesweeper game.
It consists of several classes that handle different aspects of the game: `Win_check` to verify if the player has won by checking if all non-mine cells are uncovered,
`Printing` to display the game board, `Play_Game` to manage the gameplay logic,
`Placing_mines` to randomly place mines and calculate adjacent mine counts, and `Extend_spaces` to reveal adjacent cells recursively when an empty cell is uncovered.
The main class, `Minesweeper`, initializes a 9x9 game board, places mines, and then continuously prompts the user to enter row and column coordinates to uncover cells until they either uncover a mine and lose or uncover all non-mine cells and win.Click Here

Word Puzzle


Developed a console-based Word puzzle game application using Java.
This Java program implements a simple word guessing game. It initializes a linked list to store words and their feedback.
The player repeatedly inputs 4-letter words, trying to guess a randomly selected word from a predefined list.
The program checks and counts the correct letters in the correct positions (bulls) and correct letters in the wrong positions (cows).
The results are stored and displayed after each guess. If the player guesses the word correctly, they win, and the game ends. The linked list is used to track and display.
